Regex Quick Reference

Anchors

^ - Start of string
$ - End of string
\b - Word boundary

Character Classes

\d - Digit
\w - Word character
\s - Whitespace
. - Any character

Quantifiers

* - 0 or more
+ - 1 or more

? - 0 or one
{n,m} - n to m times

Groups

( ... ) - Capture group
(?: ... ) - Non-capture group
[abc] - Character set
